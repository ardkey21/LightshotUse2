Lightshot para Windows

Você quer capturar coisas fora de seu navegador? Escolha a opção de download do Lightshot mais funcional para receber esta oportunidade. Este programa permite que você capture a tela diretamente de seu desktop.

Lightshot para Mac OS

Instale Lightshot em seu sistema operacional Mac OS X mais avançado. Capture imagens na tela e compartilhe elas em Facebook ou Twitter com seu Macintosh. O atalho no Mac é ⇧⌘ 9

Instalar

Lightshot para Ubuntu

Nâo temos uma versão de Lightshot feita especificamente para Linux ainda. Mas você ainda pode usar Lightshot usando Wine. Lightshot foi testado para compatibilidade com Ubuntu Linux mas provavelmente funcona em outros.

Instalar

Google Chrome

Esse add-on irá adicionar um novo botão em Google Chrome, trazendo todas as ferramentas de Lightshot. Tire uma foto de qualquer coisa que você pode ver em sua tela, edite ou envie para a internet para compartilhar com seus amigos.
